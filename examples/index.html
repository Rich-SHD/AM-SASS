Container

Wrap all content in a data-container to scope page widths. It often makes sense to keep the number of containers to a minimum, for example 1 for the nav, 1 for main content and 1 for the footer.

<div data-container>
  <div data-row>
    <div data-col="xs-12"></div>
  </div>
</div>

Grid layout

A row and column based grid with multiple breakpoints. A value will apply to all breakpoints at and above the declared size until a higher breakpoint value is declared. For example if a column has data-col="xs-6 xl-8" the column will be 6 wide for xs, sm, md and lg breakpoints then 8 wide at xl. The number of columns is declared in the config.

<div data-row>
  <div data-col="xs-6 sm-4 lg-8"></div>
  <div data-col="xs-6 sm-4 lg-4"></div>
  <div data-col="sm-4 lg-12"></div>
</div>

Push & pull

Change the order of columns at specific breakpoints. AM-Strap is built mobile first so the order you want columns to display at the lowest breakpoint should be the order of the code.

<div data-row>
  <div data-col="sm-6" data-push="lg-6"></div>
  <div data-col="sm-6" data-pull="lg-6"></div>
</div>

Offset

Indent columns by any number of column widths. This is ideal for centralising a column which doesn't have full width.

<div data-row>
  <div data-col="sm-6" data-offset="sm-3"></div>
</div>
